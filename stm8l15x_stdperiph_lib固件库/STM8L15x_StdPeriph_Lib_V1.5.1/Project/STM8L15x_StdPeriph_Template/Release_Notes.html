STM8L15x Standard Peripherals Library Template update History

V1.5.0 / 13-May-2011

Main Changes

  • Add the support of the STM8L15x Low density devices in STVD and EWSTM8 project templates
    • Update STVD workspaces for both Cosmic and Raisonance compilers by adding 'stm8l1526-eval-ld' project for the STM8L15x Low density devices
    • Update EWSTM8 template project by adding 'stm8l1526-eval-ld' workspace for the STM8L15x Low density devices.     
      Known limitations
  • The current available version of  RIDE7 IDE doesn't support yet the STM8L15x Low density devices.

1.4.1 - 09/29/2010

  1. STM8L15x_StdPeriph_Template:  Template projects 
  •    EWSTM8 template project update

1.4.0 - 09/24/2010

  1. General
  • Rename "project" in the workspaces with the different toolchains to "stm8l1526-eval" for stm8l15x Medium Density devices 
  • Add stm8l1528-eval project in the workspaces with the different toolchains for stm8l15x High Density devices 
  • Add new folder "Misc" in different project workspaces
  • "stm8l15x_it.c" file updated with High Density devices inetrrupt vector declaration
  • "stm8l15x_it.h" file updated to use new "INTERRUPT" macro for interrupt handler declaration
  • "stm8_interrupt_vector.c" file updated High Density devices inetrrupt vector declaration
  1. STM8L15x_StdPeriph_Template:  Template projects
  • ST Visual Develop (STVD) software toolchain
    • STVD Version 4.2.0
    • Cosmic STM8 32K compiler Version 4.3.3 (or later)
    • Raisonance STM8/ST7 C compiler Version 2.30.10.0175
  • Raisonance IDE RIDE7 (RIDE) software toolchain
    • Version: RIDE7 IDE:7.30.10.0169, RKit-STM8 for 2.30.10.0175
  • IAR Embedded Workbench for STM8 IDE (EWSTM8) software toolchain
    • Version v1.10.4
